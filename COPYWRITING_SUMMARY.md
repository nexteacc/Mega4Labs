## 🎯 产品定位与目标用户

### 产品定位

**Perplexity AI**
- 对话式 AI 答案引擎
- 结合大型语言模型与实时搜索
- 提供有来源支持的准确答案

**Comet Browser**
- AI Agent 原生浏览器
- 将被动获取信息转变为主动完成任务
- 跨页情境感知与代理执行能力

**组合价值**
```
Perplexity（获取答案）+ Comet（整合信息 + 执行任务）
= 完整的知识工作流（从搜索到执行）
```

### 目标用户

**知识工作者**（Knowledge Workers）
- 创作者（内容创作、设计师）
- 产品经理（需求分析、竞品研究）
- 研究人员（学术研究、市场调研）
- 分析师（数据分析、报告撰写）

**核心需求**
- 快速获取准确信息
- 整合多源信息
- 提高工作效率
- 结构化工作流程

## 💡 核心设计原则

### 1. 简洁有力
- **标题控制在 40 字符以内**
- **副标题控制在 120 字符以内**
- **去除冗余词汇和陈词滥调**

### 2. 价值导向
- **聚焦用户利益**：Research Smarter, Work Faster
- **强调结果而非功能**：transform workflow（而非 "has features"）
- **具体化价值**：from search to execution（而非 "better"）

### 3. 清晰层级
- **Pill 标签**：品牌定位（Curated learning hub）
- **主标题**：核心价值主张（Research Smarter. Work Faster.）
- **副标题**：产品组合说明（Comet + Perplexity）
- **支持文案**：目标用户 + 质量保证

### 4. 行动导向
- **动词开头**：Learn, Master, Watch, Unlock
- **明确 CTA**：Get Invite Code（而非 "Learn More"）
- **渐进式引导**：Hero CTA → 中间 CTA → 底部 CTA

### 5. SEO 友好
- **包含产品名**：Comet, Perplexity
- **包含关键词**：AI browser, answer engine, workflow, knowledge workers
- **自然语言**：不堆砌关键词，保持可读性

## 📝 完整文案清单

### Hero 区域（首屏）

#### Pill 标签
```
英文：⭐ Featured
韩文：⭐ 추천
日文：⭐ おすすめ
中文：⭐ 精选
```

**设计理由**：
- ✅ 简洁视觉化 - emoji 五角星比文字更吸引眼球
- ✅ 避免重复 - 顶部已有 "Learning Hub"，不需要再说一次
- ✅ 强调精选 - "Featured" 暗示这是精选内容
- ✅ 国际通用 - ⭐ emoji 在所有语言中都能理解
- ✅ 轻量级 - 不抢主标题的风头，但增加视觉趣味

#### 主标题（H1）
```
英文：Research Smarter. Work Faster.
韩文：더 스마트한 리서치. 더 빠른 작업.
日文：スマートに調査。速く仕事。
中文：更智能的研究，更高效的工作
```

**特点**：
- ✅ 32 字符，简洁有力
- ✅ 对仗工整，易记
- ✅ 聚焦核心价值（研究 + 工作）

#### 副标题
```
英文：Learn Comet and Perplexity through curated video tutorials.

韩文：엄선된 비디오 튜토리얼로 Comet과 Perplexity를 배우세요.

日文：厳選されたビデオチュートリアルでCometとPerplexityを学ぶ。

中文：通过精选视频教程学习 Comet 与 Perplexity。
```

**特点**：
- ✅ "Learn" 开头，强调学习行为
- ✅ 说明学什么（Comet and Perplexity）
- ✅ 说明怎么学（curated video tutorials）
- ✅ 62 字符，从 124 字符精简 50%
- ✅ 回归学习中心的本质定位

#### 支持文案
```
英文：For knowledge workers • Updated weekly • Expert-curated
韩文：지식 근로자를 위한 • 매주 업데이트 • 전문가 선별
日文：ナレッジワーカー向け • 毎週更新 • 専門家厳選
中文：为知识工作者打造 • 每周更新 • 专家精选
```

**特点**：
- ✅ 使用 bullet points，易扫读
- ✅ 明确目标用户
- ✅ 强调内容质量和更新频率

#### CTA 按钮
```
英文：Get Invite Code
韩文：초대 코드 받기
日文：招待コードを取得
中文：获取邀请码
```

## 🌍 多语言支持

### 支持语言
- 🇺🇸 英文（English）- 默认语言
- 🇰🇷 韩文（한국어）
- 🇯🇵 日文（日本語）
- 🇨🇳 中文（简体中文）

### 翻译原则
1. **保持信息一致性**：所有语言传达相同的核心信息
2. **本地化表达**：使用符合当地习惯的表达方式
3. **长度控制**：考虑不同语言的字符长度差异
4. **文化适配**：避免文化冲突或误解

### Fallback 策略
- 如果某个语言的内容缺失，自动回退到英文
- 确保所有用户都能看到完整内容

## 🎨 视觉层级设计

### Hero 区域布局

```
┌─────────────────────────────────────────────────────────┐
│ [⭐ Featured]                        [视频卡片 1]       │ ← 顶部对齐
│                                      [视频卡片 2]       │
│ [主标题 - 大字号]                                       │
│                                                         │
│ [副标题 - 中字号]                    [视频卡片 3]       │
│                                      [视频卡片 4]       │
│ [支持文案 - 小字号]                                     │
│                                                         │
│ [CTA 按钮]                                              │
└─────────────────────────────────────────────────────────┘
```

### 字体层级

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| Pill 标签 | 14px | 600 | Accent |
| 主标题 (H1) | 48-60px | 600 | Primary |
| 副标题 | 18-20px | 400 | Secondary |
| 支持文案 | 14px | 500 | Secondary/70% |
| 模块标题 (H2) | 32-36px | 600 | Primary |
| 模块描述 | 16-18px | 400 | Secondary |

### 间距设计

**Hero 区域**：
- Pill 到标题：24px
- 标题到副标题：16px
- 副标题到支持文案：16px
- 支持文案到按钮：24px

**模块区域**：
- 模块之间：80-112px（响应式）
- 标题到描述：12px
- 描述到视频网格：32px

### 响应式适配

**桌面端（> 1024px）**：
- Hero 区域：左右两栏布局
- 视频网格：3 列（Shorts 为 4 列）

**平板端（768px - 1024px）**：
- Hero 区域：左右两栏布局
- 视频网格：2 列

**移动端（< 768px）**：
- Hero 区域：单列布局
- 视频网格：1 列

## 📊 文案效果指标

### 可衡量指标

1. **首屏停留时间**
   - 目标：> 10 秒
   - 说明用户理解了价值主张

2. **CTA 点击率**
   - Hero CTA：目标 > 15%
   - 中间 CTA：目标 > 10%
   - 底部 CTA：目标 > 20%

3. **视频播放率**
   - 目标：> 30% 的访问者至少播放 1 个视频

4. **页面跳出率**
   - 目标：< 40%
   - 说明内容吸引用户继续浏览

5. **滚动深度**
   - 目标：> 60% 的用户滚动到底部 CTA
