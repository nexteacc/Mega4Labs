# Mega 4 Labs

AI 行业领袖访谈视频聚合平台，展示 OpenAI、Cursor、DeepMind (Gemini) 和 Anthropic 四大公司领导者的访谈、演讲和见解。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

创建 `.env.local` 文件：

```bash
# YouTube API Key (用于视频抓取)
YOUTUBE_API_KEY=your_youtube_api_key_here
```

### 3. 运行开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)
