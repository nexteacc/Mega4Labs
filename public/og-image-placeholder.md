# OG Image 占位符说明

请创建一张 **1200x630px** 的图片，命名为 `og-image.jpg`，放在 `public/` 目录。

## 设计建议：

**尺寸：** 1200 x 630 像素

**配色方案：**
- 主色：#21808d（青绿色）
- 强调色：#f1c84b（金黄色）
- 背景：#e7f3f5（浅蓝绿）或白色

**内容建议：**
```
┌─────────────────────────────────────────┐
│                                         │
│   Perplexity Pro                        │
│   Learning Hub                          │
│                                         │
│   Master AI-Powered Research            │
│   50+ Video Tutorials                   │
│                                         │
│   www.perplexitypro.info               │
│                                         │
└─────────────────────────────────────────┘
```

**设计工具：**
- Figma（推荐）
- Canva
- Photoshop
- 或使用在线工具：https://www.opengraph.xyz/

**完成后：**
将图片保存为 `public/og-image.jpg`，然后删除此文件。
